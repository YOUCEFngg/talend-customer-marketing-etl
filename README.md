# Talend Customer Marketing ETL

A Talend Open Studio ETL project demonstrating data integration from multiple CSV sources, including customer, department, and marketing datasets.

## Features

- Import multiple CSV datasets
- Process customer information
- Integrate marketing data
- Department data mapping
- ETL workflow implementation

## Technologies

- Talend Open Studio for Data Integration
- Java
- CSV Files

## Project Structure

### Input Data

- Personnes3
- Marketing
- Departements2

### Main Job

- Tp3

## Workflow

1. Load customer data.
2. Load department information.
3. Load marketing data.
4. Perform data transformation and mapping.
5. Produce processed output datasets.

## Learning Objectives

- ETL process design
- Data integration
- Talend component usage
- Multi-source data processing
- Data transformation
